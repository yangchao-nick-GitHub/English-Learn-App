# 学习进度管理工具 - 小白友好使用指南

## 🎯 概述

这是一个为小白用户设计的超简单学习进度管理系统！你只需要：

1. **编辑一个简单的 JSON 文件**
2. **点击几个按钮**

就这么简单！不需要懂技术，不需要会编程。

## 📍 文件位置

在项目文件夹中找到这个文件：

```
public/user-learning-progress.json
```

这个文件就是你的学习进度配置文件！

## 📄 JSON 文件内容说明

打开文件后，你会看到这样的内容：

```json
{
  "版本": "1.0",
  "说明": "Qwerty Learner 学习进度配置文件 - 可以手动编辑此文件来改变学习进度",

  "用户信息": {
    "学习天数": 5,
    "总学习单词数": 150
  },

  "当前学习设置": {
    "当前词书": "pets3",
    "当前章节": 2,
    "当前单词索引": 5,
    "说明": "下次学习时将从第3章第6个单词开始"
  }
}
```

## 🎮 三种使用方法

### 方法 1：直接编辑 JSON 文件（最简单）

1. **用记事本或文本编辑器打开** `public/user-learning-progress.json`
2. **修改数字**：
   - `"当前章节": 2` → 改成你想要的章节号
   - `"当前单词索引": 5` → 改成你想要的单词位置
3. **保存文件**
4. **刷新浏览器页面**
5. **点击"继续学习"**

### 方法 2：使用可视化界面（最直观）

1. **打开网页版管理界面**：

   - 在浏览器中打开 `http://localhost:5194/progress-editor`
   - 或者点击侧边栏的"进度管理"

2. **修改数字**：

   - 在"当前章节"输入框输入章节号（0-97）
   - 在"单词索引"输入框输入单词位置（0-19）

3. **点击按钮**：

   - 点击"🔄 更新配置"预览
   - 点击"✅ 保存到系统"应用

4. **刷新页面查看效果**

### 方法 3：使用浏览器控制台（最灵活）

1. **按 F12 打开浏览器控制台**
2. **输入命令**：

   ```javascript
   // 跳转到PETS3第10章第5个单词
   qwertyPets3(10, 5)

   // 跳转到Coder Dict第20章开头
   qwertyCoder(20, 0)
   ```

3. **刷新页面查看效果**

## 💡 重要提示

### 关于章节和单词索引

- **章节从 0 开始**：

  - 第 1 章 = `0`
  - 第 2 章 = `1`
  - 第 10 章 = `9`

- **单词索引从 0 开始**：
  - 第 1 个单词 = `0`
  - 第 2 个单词 = `1`
  - 第 10 个单词 = `9`

### 实际例子

如果你想跳到**第 5 章第 3 个单词**：

- JSON 文件中写：`"当前章节": 4, "当前单词索引": 2`
- 因为第 5 章对应数字 4，第 3 个单词对应数字 2

## 🎯 使用场景示例

### 场景 1：我想从第 20 章开始学习

```json
"当前学习设置": {
  "当前词书": "pets3",
  "当前章节": 19,      // 第20章
  "当前单词索引": 0    // 从该章第1个单词开始
}
```

### 场景 2：我想复习第 10 章的第 15 个单词

```json
"当前学习设置": {
  "当前词书": "pets3",
  "当前章节": 9,       // 第10章
  "当前单词索引": 14   // 该章第15个单词
}
```

### 场景 3：我想切换到 Coder Dict 第 5 章

1. 修改 JSON 文件：

```json
"当前学习设置": {
  "当前词书": "coder",  // 改成coder
  "当前章节": 4,        // 第5章
  "当前单词索引": 0
}
```

2. 在网页界面选择词书并修改章节

3. 或使用控制台：`qwertyCoder(4, 0)`

## 🔧 备份和恢复

### 备份进度

1. **手动备份**：复制 `user-learning-progress.json` 文件到其他位置
2. **网页导出**：访问进度管理页面，点击"💾 导出配置文件"

### 恢复进度

1. **手动恢复**：将备份的 JSON 文件内容复制回 `public/user-learning-progress.json`
2. **网页导入**：在进度管理页面点击"📂 从文件加载"

## ❓ 常见问题

### Q: 修改后没有生效？

A: 确保你：

1. 保存了 JSON 文件
2. 刷新了浏览器页面
3. 点击了"继续学习"按钮

### Q: 输入的章节号显示无效？

A: 检查章节号是否在有效范围内：

- PETS: 0-38
- PETS-2023: 0-88
- Coder Dict: 0-33

### Q: 我修改了 JSON 文件但网页没变化？

A: 可能是浏览器缓存问题，试试：

1. 强制刷新页面 (Ctrl+F5)
2. 或者使用网页版界面直接修改

### Q: 怎么重置所有进度？

A: 在 JSON 文件中将所有词书的进度改为：

```json
{
  "章节": 0,
  "单词索引": 0,
  "已学单词数": 0,
  "完成百分比": 0.0,
  "学习状态": "未开始"
}
```
